# Ping-pong
Пинг-понг - мультиплеерная игра. Игра написана на языке Python.                                                                                                                  
Игроки управляют ракетками: первый - клавишами "W" и "S", второй - стрелками.                                                                                                      
Задача: не пропустить мяч.
